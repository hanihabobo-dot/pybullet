🧠 Key Innovation: Know-If Fluents

Instead of using a complex belief-space planner, I implemented Know-If fluents to handle partial observability within standard PDDLStream:

; Know-If pattern for partial observability obj_at_boxel(?o, ?b) ; Ground truth obj_at_boxel_KIF(?o, ?b) ; Know-If (do we know the value?) ; Derived knowledge: K(in) = KIF ∧ in ; Known IN boxel K(¬in) = KIF ∧ ¬in ; Known NOT in boxel

💡 Why This Works

  • Optimistic planning: Assume sensing will succeed
  • Replan on mismatch: If object not found, update belief and replan
  • Simple architecture: No need for complex belief-space planners like LW1
